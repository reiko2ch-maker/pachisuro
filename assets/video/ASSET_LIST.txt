# ECLIPSE DEVOURER ULTIMATE EDITION

GitHub Pages / スマホ縦画面向けの、映像対応・液晶搭載版ダークファンタジーAT機です。

## 重要
- **東京喰種の公式映像・ロゴ・キャラ画像・音源は同梱していません。**
- 既存作品の素材は著作権上NGです。
- この版は**オリジナル演出** + **ユーザー差し替え用映像スロット**で構成しています。

## 追加要素
- 液晶シネマディスプレイ追加
- 背景動画 / カットイン動画 / 液晶動画の3レイヤー
- 通常 / 赫眼 / 前兆 / CZ / BONUS / AT / 上位 / バトル / 特化 / 貫き で映像切替
- レア役成立時の短尺液晶演出
- スマホ縦画面最適化
- SAVE / LOAD / 設定変更 / 上位貫き / 一撃性能維持
