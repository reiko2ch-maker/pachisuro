# ECLIPSE DEVOURER

GitHub Pages / ローカルでそのまま動く、ダークファンタジーAT機シミュレーターです。

## できること

- 設定1〜6の変更
- 通常 / 前兆 / CZ / EP BONUS / AT / 対決 / 特化ゾーン / 上位モード
- 上位終了後の貫き待機と次回AT上位引き継ぎ
- オートプレイ
- 自動セーブ / 手動セーブ / ロード / セーブ文字列書き出し
- GitHub Pages での公開

## 使い方

### ローカルで開く

`index.html` をブラウザで開くだけで動作します。

### GitHub Pages に載せる

1. このフォルダをそのまま GitHub リポジトリにアップロード
2. `Settings > Pages` を開く
3. `Deploy from a branch` を選択
4. `main` ブランチ / `/root` を選択
5. 公開URLにアクセス

## ファイル構成

- `index.html` : 本体UI
- `styles.css` : デザイン
- `app.js` : 抽選、状態遷移、セーブ処理

## 注意

- 実在作品・実機の画像や音源は使っていません
- オリジナルの世界観・名称で構成しています
- ローカル鑑賞・研究・試作用途向けです
